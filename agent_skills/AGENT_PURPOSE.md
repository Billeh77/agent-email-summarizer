This agent automatically summarizes incoming emails and their attachments.

When you send an email to the agent's mailbox, it will:
- Read the full email body
- Extract and analyze any PDF or DOCX attachments
- Generate a structured summary with an overview, key points, attachment highlights, and action items
- Reply to you with the complete summary

This agent handles operational email-processing requests — emails where you want a summary generated and sent back. It does NOT handle billing, account changes, or general questions unrelated to email summarization.
